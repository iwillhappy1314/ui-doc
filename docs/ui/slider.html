全宽幻灯

因为全屏幻灯需要使用背景实现，空内容没有高度，且不支持高度自适应，浏览器窗口改变时，修改幻灯高度 也可以使用一张透明的png图片来撑起高度，不过要多加载一张和幻灯图片高度一样的png图片

Carousel

其实就是幻灯的变体，原理类似，很多Slider 插件都实现了这一些，甚至有些连 tab 和 Accordion 、Toogle 也做到了一起。
